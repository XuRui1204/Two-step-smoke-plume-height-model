# Two-step-smoke-plume-height-model

## Overview
This repository contains the source code and analysis scripts associated with the peer-reviewed article:

> **"Atmospheric Convection and Aerosol Absorption Boost Wildfire Smoke Injection"**
> *Geophysical Research Letters*, DOI: [10.1029/2025GL115989](https://doi.org/10.1029/2025GL115989)

## Requirements
- Python 3.x
- Dependencies can be check in scripts.
